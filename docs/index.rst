Saisir -セジール- について
-------------------------

Saisirはcsvデータを分析し、顧客の購買傾向をつかむGUIアプリケーションになります。

現在は商品の購買傾向からクラスタを作成し、各クラスタの特徴を視覚的に捉えることができます。
